001 : PNDC Homegrown Initiative

View Code

This was a large project for a local Non-Profit, the Penn-Northwest Development Corporation Homegrown Initiative. The tech stack is HTML, CSS, JS, and Bootstrap v5.2 - with Netlify BaaS providing the hosting.

The Landing Page

A Tour of the Main Pages

This was a freelance web dev project contracted by Penn-Northwest. The goal was to build a website that allows users, at a glance, to get a good idea of what the Mercer County, PA, area is all about. In addition, through hand-coded functionality, the user is able to get in contact with the people at Penn-Northwest who will help them find work - free of charge - or start a business, pay for college, etc.

The project proved to be a real technical challenge - testing the limit of what I knew how to build, and forcing me to immerse myself in the world of web development.

For a more detailed writeup of the development process of this site, and to view the code, simply head on over to this link!
