002 : Penn-Northwest Development Corporation

View Code

This project was an entire redesign of the penn-northwest website. Its front-end was built using EJS, Vanilla CSS, Bootstrap v5.2, and JavaScript. Its backend was constructed in Express.js. It is a CRUD application which features a user account registration/authentication system.

Additional technologies used include: Mongo Atlas and Mongoose as the primary databse. Axios for API interactions. Express-Session for session/cookie management. Passport.js and jsonwebtoken for auth. Stripe Connect API for managing payments. Cloudinary for photo management/delivery.

This project was the first major fullstack application I pushed to production. It represented a serious technical challenge, with many features and hurdles to overcome.

The goal of the project was to redesign/rebuild the Penn-Northwest Corporation's website from the ground up, condensing their large site into something more manageable, user-friendly, clean, and easy to navigate.

The main technical challenge was in building a fully-fledged CRUD functionality such that Admins are able to login and make changes to the site - all to the custom spec of the client. It had to be easy to use, and integrated with their extant workflow and customer/user base.
